This app uses the AccessibilityService API to lock the screen.

-----

To lock the screen, you must grant "Accessibility" permission.
Please go to **Settings** > **Accessibility** and turn **ON** "Switch to Lock Screen."
The app will not function unless this permission is granted (turned ON).

## Overview

### Lock the screen with a single tap!

Switch to the lock screen with just one tap.
Smartphone manufacturers test power buttons by pressing them tens of thousands of times, yet hardware failures—such as the button sinking in—can still occur. Is it just bad luck?
Use this app to reduce the number of times you press the power button and avoid hardware failure.

With gesture navigation becoming the default, it has become possible to handle all operations directly on the screen.
We decided to release this app based on a feature that was well-received in the past.
It allows you to switch to the lock screen with minimal actions.

- Tapping the back of the device (Pixel Quick Tap)
- Tapping the side (Xperia Side Sense)
- Screen gestures, etc.

If your device allows you to assign apps to these actions, you can configure this app to use it conveniently.
