## Overview

### Create Adaptive Icons for [Soulful Launcher](../soulful_launcher/)

**Create adaptive icons designed for Soulful Launcher!**

### App Features

**[An intuitive home screen that doesn't rely on text]**
Soulful Icon Generator allows you to freely create icons to build a unique, personalized home screen for Soulful Launcher.

**■ Supports Adaptive Display**
* **Vector Icons (SVG format):** Create vector icons that maintain image quality even when scaled up—a key feature of Soulful Launcher.
* **Standard Android Adaptive Icons:** Create adaptive icons that remain sharp and intact even at larger sizes.
* **Foreground and Background:** Create separate foreground and background layers—a requirement for adaptive icons—and combine them into a single icon.
* **Individual Parameters:** Adjust settings such as opacity, rotation angle, and scale for each element.
* **Background Color:** Set a background color for the background layer; choose from simple solid colors or gradients.

**■ Recommended for**
* Those who want to create a unique, personalized home screen using Soulful Launcher.
* Those who want to create a simple interface for seniors or beginners, making it easy to say, "Just tap this icon."

* [ICOOON MONO](https://icooon-mono.com)
